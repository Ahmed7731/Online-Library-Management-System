# 📚 Online Library Management System

## 📌 Overview

This project is an **Online Library Management System** developed using **Django**.

The system provides a modern web interface for managing library-related features such as user authentication, book browsing, and administrative functionality.

---

## 🛠️ Technologies Used

* Python
* Django
* HTML5
* CSS3
* JavaScript
* SQLite

---

## 🎯 Features

* 🔐 User Authentication (Login & Signup)
* 🏠 Home Page Interface
* 👤 User Dashboard
* 🛡️ Admin Dashboard
* 🎨 Responsive Frontend Design
* ⚡ Interactive JavaScript Functionality
* 📂 Organized Django Project Structure

## 🚀 How to Run

1. Download or clone the repository

2. Open the project folder

3. Create a virtual environment

```bash
python -m venv venv
```

4. Activate the virtual environment

### Windows

```bash
venv\Scripts\activate
```

5. Install Django

```bash
pip install django
```

6. Run migrations

```bash
python manage.py migrate
```

7. Start the development server

```bash
python manage.py runserver
```

8. Open in browser

```text
http://127.0.0.1:8000/
```
